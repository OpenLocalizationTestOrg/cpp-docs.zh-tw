# CSimpleRow::CSimpleRow
The constructor.  
  
## Syntax  
  
```  
  
      CSimpleRow(  
   DBCOUNTITEM iRowsetCur   
);  
```  
  
#### Parameters  
 `iRowsetCur`  
 [in] Index to the current rowset.  
  
## Remarks  
 Sets [m_iRowset](../../data/oledb/csimplerow-m-irowset.md) to `iRowsetCur`.

 **Header:** atldb.h
