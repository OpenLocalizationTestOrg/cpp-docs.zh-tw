# ICommandImpl::m_bCancel
Indicates whether the command is canceled.  
  
## Syntax  
  
```  
  
unsigned m_bCancel:1;  
  
```  
  
## Remarks  
 You can retrieve this variable in the **Execute** method of your command class and cancel as appropriate.  
  
## Requirements  
 **Header:** atldb.h
