# ICommandImpl::m_bCancelWhenExecuting
Indicates whether the command can be canceled when executing.  
  
## Syntax  
  
```  
  
unsigned m_bCancelWhenExecuting:1;  
  
```  
  
## Remarks  
 Defaults to **true** (can be canceled).  
  
## Requirements  
 **Header:** atldb.h
