# CRowset::Close
Releases rows and the current [IRowset](https://msdn.microsoft.com/en-us/library/ms720986.aspx) interface.  
  
## Syntax  
  
```  
  
void Close( ) throw( );  
  
```  
  
## Remarks  
 This method releases all rows currently in the rowset.  
  
## Requirements  
 **Header:** atldbcli.h
