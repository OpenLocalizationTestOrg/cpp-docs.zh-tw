# CSession::Close
Closes the session, which was opened by [CSession::Open](../../data/oledb/csession-open.md).  
  
## Syntax  
  
```  
  
void Close( ) throw( );  
  
```  
  
## Remarks  
 Releases the **m_spOpenRowset** pointer.  
  
## Requirements  
 **Header:** atldbcli.h
