# CSession::Abort
Terminates the transaction.  
  
## Syntax  
  
```  
  
      HRESULT Abort(   
   BOID* pboidReason = NULL,   
   BOOL bRetaining = FALSE,   
   BOOL bAsync = FALSE    
) const throw( );  
```  
  
#### Parameters  
 See [ITransaction::Abort](https://msdn.microsoft.com/en-us/library/ms709833.aspx) in the *OLE DB Programmer's Reference*.  
  
## Return Value  
 A standard `HRESULT`.  
  
## Requirements  
 **Header:** atldbcli.h
