# CDynamicAccessor::GetColumnCount
Retrieves the number of columns.  
  
## Syntax  
  
```  
  
DBORDINAL GetColumnCount( ) const throw( );  
  
```  
  
## Return Value  
 The number of columns retrieved.  
  
## Requirements  
 **Header:** atldbcli.h
