# IRowsetChangeImpl::DeleteRows
Deletes rows from the rowset.

```  
   HCHAPTER /* hReserved */,  
   DBCOUNTITEM cRows,  
   const HROW rghRows[],  
   DBROWSTATUS rgRowStatus[]

```
