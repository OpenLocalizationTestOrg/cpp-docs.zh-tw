# Resource Compiler Fatal Error RC1022
expected '#endif'  
  
 An `#if`, **#ifdef**, or **#ifndef** directive was not terminated with an `#endif` directive.  
  
 Make sure that there is an `#if`, **#ifdef**, or **#ifndef** statement in effect before this statement.
