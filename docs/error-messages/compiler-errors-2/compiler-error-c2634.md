# Compiler Error C2634
'&class::member' : pointer to reference member is illegal  
  
 A pointer to a reference member is declared.  
  
 The following sample generates C2634:  
  
```  
// C2634.cpp  
int mem;  
struct S {  
   S() : rf(mem) { }  
   int &rf;  
};  
int (S::*pdm) = &S::rf;   // C2634  
```
