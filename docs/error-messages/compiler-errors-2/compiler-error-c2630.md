# Compiler Error C2630
'symbol' found in what should be a comma-separated list  
  
 The symbol appears in a context that requires a comma.  
  
 The following sample generates C2630:  
  
```  
// C2630.cpp  
// compile with: /c  
struct D {  
   D(int);  
};  
  
struct E {  
   E(int);  
};  
  
class C : public D, public E {  
   C();  
};  
  
C::C() : D(0) ; E(0) { }   // C2630  
C::C() : D(0), E(0) {}   // OK  
```
