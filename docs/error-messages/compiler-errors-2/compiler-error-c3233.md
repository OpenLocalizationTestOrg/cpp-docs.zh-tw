# Compiler Error C3233
'type' : generic type parameter already constrained  
  
 It is not valid to constrain a generic parameter in more than one `where` clause.  
  
 The following sample generates C3233:  
  
```  
// C3233.cpp  
// compile with: /clr /LD  
  
interface struct C {};  
interface struct D {};  
  
generic <class T>  
where T : C  
where T : D  
ref class E {};   // C3233  
```
