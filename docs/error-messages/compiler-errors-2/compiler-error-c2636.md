# Compiler Error C2636
'identifier' : pointer to reference member is illegal  
  
 A pointer to a reference member was declared.  
  
 The following sample generates C2636:  
  
```  
// C2636.cpp  
struct S {};  
int main() {  
   int &S::*prs;   // C2636  
   int S::*prs1;   // OK  
   int *S::*prs2;   // OK  
}  
```
