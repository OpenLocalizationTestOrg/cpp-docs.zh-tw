# Compiler Error C3210
'type' : access declaration can only be applied to a base class member  
  
 A [using declaration](../../cpp/using-declaration.md) was specified incorrectly.  
  
## Example  
 The following sample generates C3210.  
  
```  
// C3210.cpp  
// compile with: /c  
struct A {  
protected:  
   int i;  
};  
  
struct B {  
   using A::i;   // C3210  
};  
  
struct C : public A {  
   using A::i;   // OK  
};  
```
