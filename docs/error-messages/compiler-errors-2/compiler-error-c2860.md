# Compiler Error C2860
'void' cannot be an argument type, except for '(void)'  
  
 Type `void` cannot be used as an argument type with other arguments.  
  
 The following sample generates C2860:  
  
```  
// C2860.cpp  
// compile with: /c  
void profunc1(void, int i);   // C2860  
void func10(void);   // OK  
```
