# Compiler Error C2137
empty character constant  
  
 The empty character constant ( ' ' ) is not permitted.  
  
 The following sample generates C2137:  
  
```  
// C2137.cpp  
int main() {  
   char c = '';   // C2137  
   char d = ' ';   // OK  
}  
```
