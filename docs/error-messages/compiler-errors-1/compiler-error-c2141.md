# Compiler Error C2141
array size overflow  
  
 An array exceeds the 2GB limit. Reduce the size of the array.  
  
## Example  
 The following sample generates C2141.  
  
```  
// C2141.cpp  
// processor: IPF  
class A {  
   short m_n;  
};  
  
int main()  
{  
   A* pA = (A*)(-1);  
   pA = new A[0x8000000000000001];   // C2141  
  
   A* pA2 = (A*)(-1);  
   pA2 = new A[0x80000000000000F];   // OK  
}  
```
