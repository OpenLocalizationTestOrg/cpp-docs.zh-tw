# Compiler Error C2324
'identifier' : unexpected to the right of 'name'  
  
 A destructor is called using an incorrect identifier.  
  
 The following sample generates C2324:  
  
```  
// C2324.cpp  
class A {};  
typedef A* pA_t;  
int i;  
  
int main() {  
   pA_t * ppa = new pA_t;  
   ppa->~i;   // C2324  
   ppa->~pA_t();   // OK  
}  
```
