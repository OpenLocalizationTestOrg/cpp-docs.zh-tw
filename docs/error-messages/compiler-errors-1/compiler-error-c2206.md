# Compiler Error C2206
'function' : typedef cannot be used for function definition  
  
 A `typedef` is used to define a function type.  
  
 The following sample generates C2206:  
  
```  
// C2206.cpp  
typedef int functyp();  
typedef int MyInt;  
functyp func1 {};   // C2206  
int main() {  
   MyInt i = 0;   // OK  
}  
```
