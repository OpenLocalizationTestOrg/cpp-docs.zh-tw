# POINT Structure1
The **POINT** structure defines the x*-* and y-coordinates of a point.  
  
## Syntax  
  
```  
typedef struct tagPOINT {  
    LONG x;  
    LONG y;  
} POINT;  
```  
  
#### Parameters  
 *x*  
 Specifies the x-coordinate of a point.  
  
 *y*  
 Specifies the y-coordinate of a point.
