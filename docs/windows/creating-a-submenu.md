# Creating a Submenu
### To create a submenu  
  
1.  Click the menu command for which you want to create a submenu.  
  
2.  In the **New Item** box that appears to the right, type the name of the new menu command. This new command will appear first on the submenu menu.  
  
3.  Add additional menu commands to the submenu menu.  
  
 For information on adding resources to managed projects, please see [Resources in Applications](http://msdn.microsoft.com/Library/8ad495d4-2941-40cf-bf64-e82e85825890) in the *.NET Framework Developer's Guide.*  
  
 **Requirements**  
  
 Win32
