# HStringReference::Get Method
Retrieves the value of the underlying HSTRING handle.  
  
## Syntax  
  
```  
HSTRING Get() const throw()  
```  
  
## Return Value  
 The value of the underlying HSTRING handle.  
  
## Requirements  
 **Header:** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
