# SRWLockSharedTraits::Unlock Method
Releases exclusive control of the specified SRWLock object.  
  
## Syntax  
  
```  
inline static void Unlock(  
   _In_ Type srwlock  
);  
```  
  
#### Parameters  
 `srwlock`  
 A handle to an SRWLock object.  
  
## Return Value  
  
## Requirements

 **Namespace:** Microsoft::WRL::Wrappers::HandleTraits
