# IsSame Structure
Supports the WRL infrastructure and is not intended to be used directly from your code.  
  
## Syntax  
  
```  
template <  
   typename T1,  
   typename T2  
>  
struct IsSame;  
template <  
   typename T1  
>  
struct IsSame<T1, T1>;  
```  
  
#### Parameters  
 `T1`  
 A type.  
  
 `T2`  
 Another type.  
  
## Remarks  
 Tests whether one specified type is the same as another specified type.
