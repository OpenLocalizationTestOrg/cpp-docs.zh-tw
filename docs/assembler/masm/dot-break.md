# .BREAK
Generates code to terminate a [.WHILE](../../assembler/masm/dot-while.md) or [.REPEAT](../../assembler/masm/dot-repeat.md) block if *condition* is true.  
  
## Syntax  
  
```  
  
.BREAK [[.IF condition]]   
```
