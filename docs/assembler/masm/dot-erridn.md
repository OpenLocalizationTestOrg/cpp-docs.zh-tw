# .ERRIDN]
Generates an error if the text items are identical.  
  
## Syntax  
  
```  
  
.ERRIDN[[I]] <textitem1>, <textitem2> [[, message]]  
```  
  
## Remarks  
 If **I** is given, the comparison is case insensitive.
