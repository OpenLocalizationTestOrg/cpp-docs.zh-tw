# XMMWORD
Used for 128-bit multimedia operands with MMX and SSE (XMM) instructions.  
  
## Syntax  
  
```  
XMMWORD  
```  
  
## Remarks  
 `XMMWORD` is intended to represent the same type as [__m128](../../cpp/m128.md).  
  
## Example  
  
```  
movdqa   xmm0, xmmword ptr [ebx]  
```
