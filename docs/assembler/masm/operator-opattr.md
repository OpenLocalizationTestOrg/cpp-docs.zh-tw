# operator OPATTR
Returns a word defining the mode and scope of *expression*. The low byte is identical to the byte returned by [.TYPE](../../assembler/masm/operator-dot-type.md). The high byte contains additional information.  
  
## Syntax  
  
```  
  
OPATTR expression  
```
