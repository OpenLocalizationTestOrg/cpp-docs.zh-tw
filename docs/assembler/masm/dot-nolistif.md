# .NOLISTIF
Suppresses listing of conditional blocks whose condition evaluates to false (0).  
  
## Syntax  
  
```  
  
.NOLISTIF  
  
```  
  
## Remarks  
 This is the default. Same as [.SFCOND](../../assembler/masm/dot-sfcond.md).
