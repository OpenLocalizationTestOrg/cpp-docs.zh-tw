# .CONST
When used with [.MODEL](../../assembler/masm/dot-model.md), starts a constant data segment (with segment name CONST).  
  
## Syntax  
  
```  
  
.CONST  
  
```  
  
## Remarks  
 This segment has the read-only attribute.
