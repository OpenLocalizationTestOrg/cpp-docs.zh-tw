# Remainders
**ANSI 3.3.5** The sign of the remainder on integer division  
  
 The sign of the remainder is the same as the sign of the dividend. For example,  
  
```  
 50 / -6 == -8  
 50 % -6 ==  2  
-50 /  6 == -8  
-50 %  6 == -2  
```
