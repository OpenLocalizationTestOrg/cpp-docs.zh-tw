# A.7   Using the reduction Clause
The following example demonstrates the reduction clause ([Section 2.7.2.6](../../parallel/openmp/2-7-2-6-reduction.md) on page 28):  
  
```  
#pragma omp parallel for private(i) shared(x, y, n) \  
                         reduction(+: a, b)  
    for (i=0; i<n; i++) {  
        a = a + x[i];  
        b = b + y[i];  
    }  
```
