# _FREEENTRY, _USEDENTRY
## Syntax  
  
```  
#include <malloc.h>  
```  
  
## Remarks  
 These constants represent values assigned by the `_heapwalk` routines to the **_useflag** element of the **_HEAPINFO** structure. They indicate the status of the heap entry.
