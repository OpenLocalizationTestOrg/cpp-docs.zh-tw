# search_n (STL/CLR)
Searches for the first subsequence in a range that of a specified number of elements having a particular value or a relation to that value as specified by a binary predicate.  
  
## Syntax  
  
```  
template<class _FwdIt1, class _Diff2, class _Ty> inline  
    _FwdIt1 search_n(_FwdIt1 _First1, _FwdIt1 _Last1,  
        _Diff2 _Count, const _Ty& _Val);  
template<class _FwdIt1, class _Diff2, class _Ty, class _Pr> inline  
    _FwdIt1 search_n(_FwdIt1 _First1, _FwdIt1 _Last1,  
        _Diff2 _Count, const _Ty& _Val, _Pr _Pred);  
```  
  
## Remarks  
 This function behaves the same as the STL function `search_n`. For more information, see [search_n](http://msdn.microsoft.com/Library/8daa0c12-bac4-4cef-9213-45ef9a8d86af).  
  
## Requirements  
 **Header:** \<cliext/algorithm>  
  
 **Namespace:** cliext
